Commands - Math

Usage Key#

For commands that accept input, you'll see keys with the following:

  • <required> - parameters between <> must be included with the command
  • [optional] - parameters between [] are optional parameters and may be excluded
  • a | or | b - | means that you can use the parameters on either side of the symbol
  • repeat... - means that you can provide multiple parameters as input

These may be nested. Example: [parameter1 <parameter2\>] means that the input is optional, but if you provide parameter1 then parameter2 is required. [parameter...] means that input is optional, and that the input can be any number of parameters.

Commands#

cm#

Converts Sensitivity to cm/rev

Usage#

/cm <sens> <game | yaw value> <cpi>

Supported Flags#

-dp <output decimal places>

Examples#

Anima04/07/2021
/cm 3 ow 1600
FPSMathBot04/07/2021
@Anima, 28.86 cm/rev
Anima04/07/2021
/cm 2 0.022 800 -dp 8
FPSMathBot04/07/2021
@Anima, 25.97727273 cm/rev

Aliases#

cm/rev, cm/360

fov#

Finds the true vertical and horizontal FOVs for certain aspect ratio that the game is being rendered at and game/FOV scaling method(FILM notation)

To learn about FILM notation click here

Usage#

/fov <fov> <fov> <gamename | FILM notation> <aspect ratio>

Supported Flags#

-dp <output decimal places>

Examples#

Anima04/07/2021
/fov 90 4ML3 16:19
FPSMathBot04/07/2021
@Anima, Horizontal FOV: 64.551
Vertical FOV: 73.740
Anima04/07/2021
/fov 103 ow 1920:1440
FPSMathBot04/07/2021
@Anima, Horizontal FOV: 103.000
Vertical FOV: 56.631

Aliases#

fov-scaling, film, fovscailing

sens#

Converts cm/rev(default), deg/mm, inch/rev, mpi to a game sensitivity

Usage#

sens <cm/rev | deg/mm | inch/rev | mpi > <game | yaw> <cpi>

Supported Flags#

-cm // if the input value's unit is cm/rev it is the default and optional
-deg // if the input value's unit is deg/mm
-inch // if the input value's unit is inch/rev
-mpi // if the input value's unit is mpi

Examples#

Anima04/07/2021
/sens 28.5 quake 1600
FPSMathBot04/07/2021
@Anima, 0.91148

Aliases#

sens-cm, sens-deg, sens-inch, senscm, sensdeg, sensinch, sensmpi, sens-mpi
