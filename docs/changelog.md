## v2.10 - 4/20/2021

Moved the whole project to a the akairo framework instead of commando because it lacks a lot of features that are helpful

### Added commands

- [fovconvert](./commands/math.mdx#fovconvert): Converts fov between games or film notations relative to aspect ratio
- [arcmin](./commands/math.mdx#arcmin): Convert a sensitivity value to arcmin(minute of arc per inch)

### Added Supported games

- Totally Accurate Battlegrounds
- PlayersUnkown: Battleground
- Palidins
